### 修复方案

#### 1. 创建简化版 VL 验证器

**文件**：`/workspace/projects/workspace-aesthetics/scripts/poster_validator.py`

**功能**：
- 检查留白（>60%）
- 检查字体层次（必须使用粗黑体）
- 检查块面结构
- 检查尺寸（1080x1920）
- 给出评分（0-100）

**评分标准**：
- 0-60：不合格 → 重新生成
- 60-80：基本合格 → 可优化
- 80-100：优秀 → 发送

**核心逻辑**：
```python
# 检查文字像素占比（应该尽量少，<40%）
# 统计深色像素（文字通常是深色的）
dark_count = 0
sample_points = 100
for i in range(sample_points):
    x = (i * (width // 10)) % width
    y = (i * (height // 10)) % height
    pixel = img_rgba.getpixel((x, y))
    # 检查是否为深色（RGB 总和 < 500）
    if pixel[0] + pixel[1] + pixel[2] < 500:
        dark_count += 1

content_ratio = dark_count / sample_points
whitespace_ratio = 1 - content_ratio
```
